📖 Strassen's Matrix Multiplication

1. Introduction: Why do we need it?

Imagine you have two matrices, A and B, both of size N × N. If you multiply them using the standard row-by-column method you learned in school, you have to do N3 multiplications.

  • For a 2 × 2 matrix, that's 23 = 8 multiplications.
  • For a 1000 × 1000 matrix, that's 1,000,000,000 multiplications!

In 1969, a mathematician named Volker Strassen shocked the math world by proving that O(N3) is not the fastest way to multiply matrices. He found a clever way to reduce the number of multiplications, speeding up the algorithm significantly for large matrices.

2. The Core Concept: Divide and Conquer

Strassen's algorithm is a classic Divide and Conquer approach. We take our large N × N matrices and divide them into 4 smaller submatrices, each of size N/2 × N/2.

Let A and B be our matrices, and C be the result (C = A × B):

[ C11   C12 ]
[ C21   C22 ]
=
[ A11   A12 ] × [ B11   B12 ]
[ A21   A22 ]      [ B21   B22 ]

In the Standard Divide and Conquer method, we would calculate C like this:

C11 = A11B11 + A12B21
C12 = A11B12 + A12B22
C21 = A21B11 + A22B21
C22 = A21B12 + A22B22

Notice how many times we multiply two submatrices together? Exactly 8 times. This still gives us O(N3) time complexity.

Strassen's "Magic" Trick:
Strassen found a highly clever combination of additions and subtractions that allows us to find all four quadrants of C using only 7 matrix multiplications instead of 8! Trading 1 multiplication for a few extra additions is a huge win because matrix multiplication is extremely slow compared to matrix addition.

3. The 7 Formulas (P1 to P7)

Here are the 7 products Strassen discovered. (Don't worry about memorizing them unless your professor specifically asks you to; focus on understanding the concept!)

Step 1: Calculate the 7 Products

P1 = A11 × (B12 - B22)
P2 = (A11 + A12) × B22
P3 = (A21 + A22) × B11
P4 = A22 × (B21 - B11)
P5 = (A11 + A22) × (B11 + B22)
P6 = (A12 - A22) × (B21 + B22)
P7 = (A11 - A21) × (B11 + B12)

Step 2: Combine them into Matrix C

Once you have P1 through P7, you construct the final matrix using simple addition and subtraction:

C11 = P5 + P4 - P2 + P6
C12 = P1 + P2
C21 = P3 + P4
C22 = P5 + P1 - P3 - P7

4. Time Complexity Analysis

Since we divide the problem into 7 subproblems (instead of 8), each of size N/2, our recurrence relation becomes:

T(N) = 7T(N/2) + O(N2)

Where O(N2) is the time taken to do the matrix additions and subtractions. Using the Master Theorem, we compare Nlogba to f(N):

  • a = 7, b = 2 ⇒ Nlog27 ≈ N2.81
  • f(N) = O(N2)

Since N2.81 > N2, the time complexity is:

O(Nlog27) ≈ O(N2.81)

This is significantly faster than O(N3) for huge numbers!
