The N-Queens Problem

Visualizing Backtracking for N = 4

1. Problem Definition

The N-Queens puzzle is the problem of placing N chess queens on an N × N chessboard so that no two queens threaten each other.

For N = 4, we must place 4 queens on a 4×4 board following these strict rules:

  • No two queens can share the same row.
  • No two queens can share the same column.
  • No two queens can share the same diagonal.

2. The Backtracking Approach

Backtracking is an algorithmic technique for solving problems recursively by trying to build a solution incrementally, one piece at a time, and removing those solutions that fail to satisfy the constraints of the problem at any point of time.

Algorithm Logic (Row by Row)

  1. Start in the leftmost column of the current row (Row 1).
  2. If all queens are placed, return true (Solution found).
  3. Try all columns (1 to 4) in the current row. For each column:
    • If placing the queen is safe (no conflicts), place the queen and recursively check the next row.
    • If checking the next row leads to a solution, return true.
    • If it leads to a dead end, backtrack (remove the queen) and try the next column.
  4. If all columns have been tried and no safe spot is found, return false to trigger backtracking in the previous row.
