The Partition Rule

\(A[P] > Pivot \rightarrow Stop\)

P scans right to find values that belong in the right sub-array.

\(A[Q] \le Pivot \rightarrow Stop\)

Q scans left to find values that belong in the left sub-array.

Complexity Breakdown

AVERAGE CASE

O(n log n)

Occurs when pivot splits array into equal halves. Efficiency is maximized.

WORST CASE

O(n²)

Occurs on already sorted arrays. Pivot splits are heavily unbalanced (0 and n-1).
